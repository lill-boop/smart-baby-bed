# ESP32-S3 SimpleFOC 平衡小车调参指南

## 1. 串口命令格式
波特率: **115200**
结束符: **Newline (\n)**

| 命令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `C` | **校准零点** (Calibrate)。**必须在小车直立时发送！** | `C` |
| `E` | **启动** (Enable) | `E` |
| `D` | **停止** (Disable) | `D` |
| `S` | 切换方向 (Switch Dir) | `S` |
| `BP<值>` | 设置平衡 **P** 参数 (Proportional) | `BP2.5` |
| `BI<值>` | 设置平衡 **I** 参数 (Integral) | `BI0.0` |
| `BD<值>` | 设置平衡 **D** 参数 (Derivative) | `BD0.2` |
| `?` | 查看当前状态 | `?` |

## 2. 调参步骤 (Tuning Guide)

### 第一阶段：寻找平衡点 (P)
1. 将 **I** 和 **D** 设为 0 (`BI0`, `BD0`)。
2. 将 **P** 设为较小值 (如 `BP1.0`)。
3. 启动小车 (`E`)。
4. 如果小车**软绵绵**直接倒下 -> **增加 P** (`BP1.5`, `BP2.0`...)。
5. 如果小车**高频震荡/抖动** -> **减小 P**。
6. **目标**：找到一个让小车能勉强站立，或者围绕平衡点前后大幅摆动的 P 值。

### 第二阶段：消除震荡 (D)
1. 保持 P 不变。
2. 慢慢增加 **D** (`BD0.05`, `BD0.1`, `BD0.15`...)。
3. **D 的作用是阻尼**，它会抑制摆动。
4. 如果 D 太大，小车会出现**高频抖动**（像帕金森一样）。
5. **目标**：找到一个 D 值，让小车被推一下后能迅速稳住，不再前后且没有高频抖动。

### 第三阶段：速度环修正 (Velocity PID) - 选做
如果小车能平衡但**容易溜车**（慢慢向一个方向漂移）：
1. 确保已校准 (`C`)。
2. 适当增加速度环的参数（这是内环），我的代码里默认是 `LP0.1`, `LI5.0`。
   - `LP0.2` (增加速度响应)
   - `LI10.0` (增加保持力)

## 3. 常见问题诊断

| 现象 | 原因 | 解决方法 |
| :--- | :--- | :--- |
| **原地前后猛烈摆动** | P太大 或 D太小 | 减小 `BP` 或 增加 `BD` |
| **高频嗡嗡响/震动** | D太大 或 P太大 | 减小 `BD` (优先) 或 减小 `BP` |
| **软绵绵，一推就倒** | P太小 | 增加 `BP` |
| **虽然平衡但慢慢移动** | 零点没校准 | 扶正后输入 `C` |
| **刚启动就疯狂飞转** | 电机方向反了 | 输入 `S` 切换方向 |
| **电机没力/推不动** | 电压限制太低 | 代码中 `voltage_limit` 需调高 |

## 4. 当前默认参数 (Stable Version)
- **Balance P**: 2.5
- **Balance I**: 0.0
- **Balance D**: 0.1
- **Motor Voltage**: 10V
